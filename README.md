# social-plugin

App for streaming facebook feeds

## Run App

```
npm install
npm run dev
```

## Run App via :

Docker
```
docker run -p [PORT]:8080 aditiapratamagg/social-plugin
```

Docker-Compose
```
docker-compose up --build
```

## Run Test!

running test with `jest`
```
npm run test
```
